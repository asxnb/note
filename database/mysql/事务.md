# 事务操作

查看是否自动提交(1 自动 | 0 手动)

```sql
select @@autocommit
```

设置为手动提交(不建议开启,需要的时候手动开启事务即可)

```sql
set @@autocommit = 0
```

开启事务

```sql
start transaction
或
begin
```

提交当前事务

```sql
commit
```

回滚当前事务

```sql
rollback
   ```
