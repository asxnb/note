## MySQL

```sh
docker run -p 3306:3306 -d --name mysql \
                    -e MYSQL_ROOT_PASSWORD=lide \
                    -v/conf_path:/etc/mysql \
                    -v/lib_path:/var/lib/mysql \
                    -v/log_path:/var/log \
                    mysql:8
```

## portainer

```sh
docker run -d --network=host --name portainer --restart=always -v /var/run/docker.sock:/var/run/docker.sock portainer/portainer-ce
```
